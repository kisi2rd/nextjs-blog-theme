책 이름: 1. Html + Css + 자바스크립트 / 저자: 김기수

책을 읽게된 계기: 이 책에 코딩이외에도 Html하고 Css가 웹사이트하고
관련이 있다는 것에 흥미를 느꼈고 진로가 소프트웨어과를 목표로 하고있어서
이 책에 더 관심이 갔던 것 같습니다.

줄거리: 이 책은 웹사이트를 배우는 Html 하고 Css, 자바스크립트를 배우는 책이고
우리가 흔히아는 웹사이트들이 거의 저 코딩을 기반으로 만들어진다고 합니다.
이 책에서는 웹 사이트의 기본 구조와 어떻게 기능하는지에 대해 알려준 책이다.

책 이름: 2. 악성코드 / 저자: 문성호

책을 읽게된 계기: 해킹과 보안에 관한 소설 책이여서 제 진로하고도
관련이 있지 않을까라는 생각이 들었고 코딩에 관한 소설 책이 많지 않았지만
악성코드에 관해 재밌게 알거 같아서 이 책을 선정하게 되었습니다.

줄거리: 이 책은 여자 연예인을 해킹해 결국 죽음으로 몰고 간 해커가 회심한 후에
국민의 모든 정보를 감시하려는 해커들과 맞서 싸우는 과정을 그린 퓨전 IT소설 입니다.
소설을 보면 해커의 윤리의식과 일반인의 사이버 안전의식이 왜 중요한지가 들어있는 책이였습니다.

책 이름: 3. 포텐의 정보보안 카페

책을 읽게된 계기: 정보보안에 대해서 조금 더 자세하게 알 수있지 않을까 생각했었고
정보보안이 어렵고 복잡하기만한 내용이 아니라는 것을 알게 해줄거 같아서
이 책을 고르게 되었습니다.

줄거리: 4차산업의 시대 정보보안의 기본지식을 알려주는 책이였고
이 책을 읽어보니 암호부터 엔터프라이즈 보안까지 정보보안의 기초를 담아둔 책이여서 알기 쉬웠고
내가 부족한 부분이 이 책에 있어 배워갈 수 있었던 것 같습니다.
